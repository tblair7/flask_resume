Gathering Steam

Gathering Steam utilizes sentiment analysis of Steam game reviews as a function of time to provide game developers an early alert to predicted changes in game ratings and topics that are related to these predictions such that the developers can make informed decisions about additional game changes.


Example 1: ARK Example 2: No Man's Sky Example 3: Monster Hunter World
